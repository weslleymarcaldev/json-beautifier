# JSON Beautifier & Converter

Format, minify and convert JSON directly in VS Code.

## Full Manual

For complete documentation (installation, usage, troubleshooting and development), see [MANUAL.md](MANUAL.md).

## Public Landing

For a short commercial presentation page, see [landing-publica.html](landing-publica.html).

## Features

- **Format JSON** — Pretty print with 2-space indentation
- **Minify JSON** — Remove all whitespace
- **Convert to YAML** — Transform JSON into YAML format

## Usage

Open the Command Palette (`Ctrl+Shift+P`) and type:

- `JSON: Format`
- `JSON: Minify`
- `JSON: Convert to YAML`

Or use the shortcut `Ctrl+Shift+F` to format instantly.
