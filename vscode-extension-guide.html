JSON Beautifier & Converter · Manual Completo

Extensao para VS Code que formata, minifica e converte JSON para YAML diretamente no editor. Este guia cobre instalacao, uso, exemplos, erros comuns, desenvolvimento e distribuicao.

VS Code 1.110+ 3 comandos principais Sem dependencia de runtime Publisher: weslleymarcal
README.md MANUAL.md Landing publica

1. O que esta extensao faz

O JSON Beautifier foi criado para acelerar tarefas rotineiras com JSON no VS Code, sem sair do editor.

  • formata JSON com indentacao de 2 espacos
  • minifica JSON removendo espacos e quebras
  • converte JSON para YAML simples
  • processa selecao ou arquivo inteiro

2. Para que usar

Leitura e depuracao Use o formatador para entender payloads de API e revisar configuracoes.
Reducao de tamanho Use minify para enviar JSON compacto para integracoes e variaveis de ambiente.
Migracao para YAML Converta rapidamente estruturas JSON para arquivos de configuracao simples.
Produtividade no editor Evita alternar entre sites e ferramentas externas para tarefas basicas.

3. Requisitos

  • VS Code 1.110.0 ou superior
  • Entrada em JSON valido para todos os comandos
  • Permissao de edicao no arquivo aberto
Nota: a extensao roda localmente no editor e nao depende de API externa para processar o texto.

4. Instalacao

Opcao A: Marketplace (recomendado)

  1. Abra o VS Code e va em Extensions (Ctrl+Shift+X).
  2. Pesquise por json-beautifier.
  3. Confirme o publisher weslleymarcal e clique em Install.

Opcao B: Instalacao por VSIX

  1. Abra Extensions.
  2. Clique no menu ... e escolha Install from VSIX.
  3. Selecione o arquivo .vsix e conclua.

Opcao C: Rodar localmente para desenvolvimento

# na pasta do projeto npm install npm run compile # iniciar Extension Development Host # no VS Code: tecla F5

5. Comandos e atalhos

  • JSON: Format (Pretty Print) - formata com 2 espacos
  • JSON: Minify - remove espacos e quebras desnecessarias
  • JSON: Convert to YAML - converte JSON para YAML simples

Atalho: Ctrl+Shift+F (Windows/Linux) e Cmd+Shift+F (macOS) para o comando de formatar.

Atencao: o atalho pode conflitar com atalhos de busca do VS Code ou do seu setup. Se precisar, remapeie em Keyboard Shortcuts.

6. Como usar no dia a dia

  1. Abra um arquivo com JSON.
  2. Opcional: selecione apenas o trecho que deseja transformar.
  3. Abra a Command Palette (Ctrl+Shift+P).
  4. Execute um dos comandos JSON da extensao.
  5. Confira o resultado no editor.
Regra de escopo: com selecao ativa, so a selecao e alterada. Sem selecao, o arquivo inteiro e substituido.

7. Exemplos reais

Exemplo A: formatar

Entrada {"name":"Ana","age":27,"skills":["js","ts"]} Saida { "name": "Ana", "age": 27, "skills": [ "js", "ts" ] }

Exemplo B: minificar

Entrada { "name": "Ana", "active": true } Saida {"name":"Ana","active":true}

Exemplo C: converter para YAML

Entrada { "name": "Ana", "skills": ["js", "ts"], "active": true } Saida name: Ana skills: - js - ts active: true

8. Comportamento interno

Fluxo simplificado quando voce executa um comando:

  1. a extensao captura texto da selecao ou documento inteiro
  2. executa JSON.parse para validar e transformar em objeto
  3. aplica a transformacao selecionada
  4. substitui o trecho no editor com o resultado
  5. mostra mensagem de sucesso ou erro
Comando interno (ids): json-beautifier.format json-beautifier.minify json-beautifier.toYaml

9. Erros e mensagens

  • sucesso: JSON Beautifier: done!
  • falha: JSON invalido: ...

Erros mais comuns:

  • virgula sobrando no fim de objeto/array
  • chaves sem aspas duplas
  • comentarios em JSON
  • trecho selecionado incompleto

10. Limitacoes

  • exige JSON valido para qualquer operacao
  • nao converte automaticamente outros formatos para JSON
  • conversao para YAML e basica, voltada para cenarios comuns
  • nao processa lotes de arquivos automaticamente
Atencao: em casos de YAML avancado (anchors, tags customizadas, formatos especiais), valide manualmente o resultado.

11. Solucao de problemas

Comando nao aparece na palette

Recarregue a janela do VS Code e confirme que a extensao esta instalada/habilitada.

Nada acontece ao executar comando

Verifique se ha um editor de texto ativo e se o conteudo e JSON valido.

Erro ao testar localmente com F5

Rode npm run compile antes de iniciar e confira erros no painel Problems.

Atalho conflitando

Abra Keyboard Shortcuts e redefina o binding do comando json-beautifier.format.

14. FAQ

Posso usar em parte do arquivo?

Sim. Selecione o trecho antes de executar o comando.

A extensao altera o arquivo original?

Sim, o texto do editor ativo e substituido. Use Undo (Ctrl+Z) se necessario.

Precisa internet para funcionar?

Nao para formatar, minificar ou converter. O processamento e local.

Serve para JSON muito grande?

Funciona, mas em arquivos muito extensos pode haver atraso perceptivel por limite do editor/maquina.
