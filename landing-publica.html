JSON Beautifier

Formate, minifique e converta JSON para YAML em segundos, sem sair do VS Code. Feito para quem trabalha com APIs, configs e payloads todos os dias.

Ver manual completo README MANUAL

Formato legivel

Pretty print com 2 espacos para leitura tecnica e revisao rapida.

JSON enxuto

Minify instantaneo para payloads menores e envio mais limpo.

JSON para YAML

Conversao pratica para casos comuns de configuracao.

Como comecar em 1 minuto

1) Instale no VS Code (Extensions -> buscar "json-beautifier") 2) Abra um JSON 3) Ctrl+Shift+P e rode: - JSON: Format (Pretty Print) - JSON: Minify - JSON: Convert to YAML
